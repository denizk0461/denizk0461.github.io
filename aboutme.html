Tools I use

FL Studio

FL Studio is a digital audio workstation that I use for music production and audio manipulation. It's a fantastic tool that I have been using since 2014 and wouldn't want to miss.

Paint.net

Paint.net is an image editing tool that is based on Paint. It offers a greater variety of options, tools, and extension support – BoltBait's plugin pack is a must-have. I enjoy using Paint.net for quick and easy image editing, whether it's sprite creation, quick sketches, or making memes.
